📚 Library Management System

A simple and user-friendly Library Management System developed in Java Swing. This desktop application helps librarians manage books, issue books to students, return books, and view library statistics through a graphical interface.


---

🚀 Features

✅ Add new books
✅ Issue books to students
✅ Return issued books
✅ Dashboard with live statistics
✅ Search books by Book ID
✅ Input validation system
✅ Interactive GUI using Java Swing


---

🛠️ Technologies Used

Java

Java Swing

AWT

JTable & DefaultTableModel

CardLayout

Event Handling



---

📂 Project Modules

🏠 Dashboard

The dashboard displays:

Total Books

Issued Books

Total Students


It updates records dynamically and allows users to view data tables by clicking on cards.


---

➕ Add Book Module

This module allows librarians to add books into the system.

Fields:

Book ID

Book Title

Author Name


Validation:

Empty fields are not allowed

Book ID must be numeric



---

📕 Issue Book Module

This module is used to issue books to students.

Student Information:

Student Name

CMS ID

Department

Semester

Book ID

Issue Date


Features:

Checks if Book ID exists

Stores issue date automatically

Saves issued records in table format



---

🔄 Return Book Module

This module handles returning books.

Features:

Search issued books by Book ID

Return books using a button

Automatically removes returned books from records
